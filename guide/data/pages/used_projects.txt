====== used projects ======

This is a list of the various software projects integrated into Conzept (called "**apps**" in this context).

===== background services =====

  * **[[https://github.com/steve-jansen/json-proxy|json-proxy]]** - JSON CORS-proxy with support for secret-API-keys (MIT)
  * **[[https://github.com/gnuns/allOrigins|allOrigins]]** - general CORS-proxy (any file type). (MIT)
